# OpenWebUI-Stock-Analytics
An intelligent OpenWebUI-based RAG Pipeline that delivers comprehensive, data-driven stock market analysis. This pipeline integrates real-time financial data, historical market trends, and expert knowledge sources to generate detailed investment insights, fundamental analysis reports, and technical assessments.

THIS TOOL USES SQLITE & DOCKER FOR ITS RAG PIPELINE

The tool itself is simple to deploy using Docker. 
Just take the file and using OpenWebUI's GUI; paste or drop the Python file in the "Tools" Section.
If you want to manually put the file in OpenWebUI, just drop the file into the "Tools" Folder 
(Note: The API Key Pop-Up may not appear if you do this)

REMEMBER TO EDIT THE LOCATION PARAMETER FOR THE DATABASE BEFORE CALLING THE TOOL (ToDo-Adjust)

If using Docker (Reccomended)
To use the database RAG (required for operation) just edit the parameter for where the DB is stored and
adjust it to the location of the Database in your Docker instance/wherever the file is stored.
If not using Docker (Not Reccomended): Use Docker.
